# 青年公寓服务管理平台

基于 Spring Boot 和 MyBatis-Plus 开发的青年公寓服务管理平台，面向青年公寓租赁与日常管理场景，提供房屋信息管理、预约看房、投诉反馈、博客社区等功能。

系统包含用户端和管理端，支持管理员、商家、普通用户三类角色使用。

## 主要功能

- 房屋信息发布、浏览、搜索、收藏和留言
- 在线预约看房及预约订单审核
- 房屋投诉提交与处理
- 博客发布、收藏和评论互动
- 用户、商家及管理员信息管理
- 轮播图、数据字典和系统配置管理
- 基于 ECharts 的数据统计展示

## 技术栈

### 后端

- Spring Boot 2.2.2
- MyBatis-Plus
- Apache Shiro
- MySQL
- Maven

### 前端

- Vue 2
- Element UI
- Axios
- ECharts
- Layui

## 运行环境

- JDK 8+
- Maven 3.6+
- MySQL 5.7+
- Node.js 14+（修改管理端前端时需要）
